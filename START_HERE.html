üéØ Quick Start (5 Minutes)

Step 1: Test the Portfolio

Open index.html in your web browser to see the template in action.

Open Portfolio

Step 2: Essential Customizations

Open index.html in a text editor and update:

  • Your name, title, and tagline (lines 51-53)
  • Email addresses (search for "your.email@example.com")
  • LinkedIn URL (search for "linkedin.com/in/yourprofile")
  • GitHub URL (search for "github.com/yourusername")

Step 3: Add Your Content

  • About Section: Write your bio and philosophy
  • Skills: List your technical competencies
  • Projects: Add 3-6 featured projects with Problem-Solution-Impact narratives
  • Experience: Include work history and education

Step 4: Setup Images

Create an assets/images/ folder and add:

  • Professional headshot (400x400px)
  • Project images (800x500px each)

üé® Customization Tips

Change Colors

Edit CSS variables in styles.css (lines 5-23):

:root { --primary-color: #d4a574; /* Your accent color */ --secondary-color: #1a1a1a; /* Dark color */ }

Change Fonts

1. Pick fonts from Google Fonts

2. Update the font link in index.html (line 12)

3. Update CSS variables in styles.css (lines 19-20)

Setup Contact Form

Choose one of these free services:

  • FormSubmit: Visit formsubmit.co
  • Formspree: Visit formspree.io
  • Netlify Forms: Built-in if deploying to Netlify

Update script.js with your chosen service (see README for details)

üöÄ Deployment Options

Recommended: Netlify

  1. Create a GitHub account (if you don't have one)
  2. Push your code to a GitHub repository
  3. Sign up for Netlify
  4. Connect your GitHub repository
  5. Deploy with one click!

Alternative Options

  • GitHub Pages: Free hosting at yourusername.github.io
  • Vercel: Similar to Netlify, great for static sites
  • Cloudflare Pages: Fast CDN with unlimited bandwidth
üí° Pro Tip: Purchase a custom domain (yourname.com) for a more professional appearance. Most domain registrars cost $10-15/year.
