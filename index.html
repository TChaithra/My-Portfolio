Code Demo
Python React/TypeScript TFLite GStreamer NXP i.MX8M+

Smart AI IP Camera — Host Application

Problem

Deploying multiple AI video analytics use cases (helmet detection, triple riding, overspeed, driver monitoring) across edge devices required a unified platform that could manage boards, cameras, models, and streams without per-model custom infrastructure.

Solution

Built a full-stack Smart AI IP Camera platform with a React/TypeScript frontend for orchestration, a Python Flask backend as the control plane, and a universal stream process on each NXP i.MX8M+ board. The system is model-agnostic and camera-agnostic — supporting dynamic model swap, camera swap, pause/resume, GStreamer-based RTSP streaming, and NPU-accelerated TFLite inference.

Impact

Delivered a production-grade platform running 5+ AI models on edge hardware with dual-FPS monitoring (inference vs. UI), centralized AI event storage, and process isolation. New models can be added without modifying the streaming or control logic — enabling rapid deployment of new use cases.

Code Demo
C++ ARM Cortex-M4 LoRaWAN FreeRTOS

Low-Power IoT Sensor Network

Problem

Agricultural client needed a cost-effective solution to monitor soil moisture, temperature, and humidity across 50+ acres with minimal infrastructure. Traditional WiFi-based solutions required frequent battery changes and had poor range.

Solution

Designed a custom PCB featuring an ARM Cortex-M4 MCU with LoRaWAN connectivity and ultra-low-power sensor integration. Implemented adaptive sampling algorithms in C++ with FreeRTOS, enabling nodes to intelligently adjust their duty cycle based on environmental changes.

Impact

Reduced power consumption by 65% compared to baseline design, extending battery life from 6 months to 2+ years. Successfully deployed 120 sensor nodes across 3 farms, providing real-time data that improved irrigation efficiency by 30%.

Code Paper
FPGA VHDL FOC Xilinx

High-Performance BLDC Motor Controller

Problem

Robotics application required precise motor control with sub-millisecond response time and minimal jitter. Existing MCU-based solutions couldn't meet the timing requirements while handling multiple motor control loops simultaneously.

Solution

Developed an FPGA-based Field-Oriented Control (FOC) system using VHDL on Xilinx Artix-7. Implemented parallel processing architecture with custom fixed-point arithmetic units and high-speed ADC interfaces for current sensing.

Impact

Achieved control loop update rate of 50kHz with position accuracy within 0.1 degrees. Reduced motor cogging torque by 40% and improved energy efficiency by 25%. System now controls 6 motors simultaneously with deterministic performance.

Code
C DSP Audio STM32

Real-Time Audio Signal Processor

Problem

Audio equipment manufacturer needed a compact, low-latency DSP solution for professional audio applications. Requirements included 4-channel processing with configurable effects chain and latency under 5ms.

Solution

Designed hardware using STM32H7 with dual ADC/DAC configuration and optimized firmware implementing FFT-based filtering, dynamic range compression, and parametric EQ in C. Used CMSIS-DSP library with hand-optimized assembly for critical sections.

Impact

Delivered total system latency of 3.2ms while processing 96kHz/24-bit audio. Achieved THD+N of -105dB and dynamic range of 120dB. Product launched successfully with 2000+ units shipped in first year.
